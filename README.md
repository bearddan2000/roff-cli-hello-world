# roff-cli-hello-world

## Description
A POC for roff programming language.

## Tech stack
- groff
- troff

## Docker stack
- ubuntu:22.04

## To run
`sudo ./install.sh -u`

Output in `bin`.

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
